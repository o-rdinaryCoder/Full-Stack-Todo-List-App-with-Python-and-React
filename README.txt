This is a Full Stack TODO LIST App made using: ReactJS, NodeJS, CSS to design the frontend side, and Python, Flask, SQLAlchemy to construct the backedn API and database respectively. 
To run this application, simply follow the following steps inside a terminal.

- First, run backend/main.py
- Then, navigate to frontend and start the react app with the command: npm start. 

The data is stored locally inside the SQLite database named mydatabse.db (/backend/instance)
